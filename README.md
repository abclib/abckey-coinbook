## abckey-coinbook

[![NPM version](https://img.shields.io/npm/v/@abckey/coinbook.svg)](https://www.npmjs.com/package/@abckey/coinbook)


## Install

```cmd
npm i @abckey/coinbook
```

## Usage

```js
import Coinbook from '@abckey/coinbook'

Coinbook.get('btc') // Symbol
```
